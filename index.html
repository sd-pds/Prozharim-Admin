Что и куда подключать

Краткая памятка прямо в панели.

Cloudflare Worker

Вставь полный worker из архива в существующий Worker. В Variables / Secrets добавь: ADMIN_TOKEN, GH_TOKEN, GH_OWNER, GH_REPO, GH_BRANCH, BOT_TOKEN, CHAT_ID, VK_TOKEN, VK_PEER_ID, SECRET_PROZHARIM и подключи KV namespace ORDERS_KV.

GitHub

GH_OWNER — владелец репозитория, GH_REPO — имя репозитория основного сайта, GH_BRANCH — обычно main. Токену GitHub нужны права на contents: read/write.

Админка

Залей этот архив в отдельный репозиторий GitHub Pages, открой сайт на отдельном домене, введи URL Worker и ADMIN_TOKEN в боковой панели. Данные сохраняются через Worker прямо в GitHub и KV.
